DueLight Firmware Updater

v1.3.0

Instructions

  1. Plug the DueLight into your computer via USB
  2. Click Install Firmware below
  3. Select the USB serial port when prompted
  4. Wait for the install to complete (~30 seconds)
Browser not supported. Your browser does not support WebSerial. Please use Google Chrome or Microsoft Edge (version 89+) on a desktop computer. School-managed Chromebooks may have this feature disabled by an administrator.
Serial access blocked. Your browser or device policy is preventing access to USB serial ports. This is common on school-managed Chromebooks. You may need to flash the device on a different computer.

What's new in v1.3.0

  • WiFi scan now retries automatically if it fails
  • You can now type a network name manually if scanning doesn't find it
  • Firmware version and diagnostic links shown during setup
  • Re-scan button added to WiFi setup

Note: This requires a Chrome-based browser with WebSerial support. If the button above doesn't work, the firmware can be installed via USB at school.
